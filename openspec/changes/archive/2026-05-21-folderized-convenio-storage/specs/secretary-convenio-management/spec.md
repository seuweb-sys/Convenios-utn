# Delta for Secretary Convenio Management

## ADDED Requirements

### Requirement: Folder-based Document Generation for Secretary

The system MUST create a dedicated Drive folder by default for convenios generated by secretaries, placing the `.docx` inside it.

#### Scenario: Secretary submits a new convenio
- GIVEN a secretary submits a new convenio application
- WHEN the document generation is triggered
- THEN the system MUST create a new Drive folder
- AND place the generated `.docx` inside this folder

### Requirement: Secretary Status-Move URL Detection

The system MUST process status moves triggered by secretary actions handling both folder (`/folders/`) and legacy single-file (`/d/`) Drive links correctly.

#### Scenario: Secretary action triggers a move on a folderized convenio
- GIVEN a convenio's Drive URL contains `/folders/`
- WHEN a secretary action triggers a move (e.g., sending for review)
- THEN the system MUST move the entire folder to the appropriate directory

#### Scenario: Secretary action triggers a move on a legacy convenio
- GIVEN a convenio's Drive URL contains `/d/`
- WHEN a secretary action triggers a move
- THEN the system MUST move the single file to the appropriate directory
